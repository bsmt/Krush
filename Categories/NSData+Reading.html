Instance Methods

intDataInRange:

Convert the data in the specified range to an unsigned long

- (unsigned long)intDataInRange:(NSRange)range

Parameters

range

The range to convert the data from.

Return Value

The data converted to an unsigned long. <00001000> becomes 0x000010000.

Discussion

Equivalent of [[data subDataWithRange:range] toInt]; Note: Limited to 8 bytes or less because it returns as an unsigned long.
