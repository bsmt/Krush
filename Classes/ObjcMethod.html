methodFromStruct:inMachO:type:

Create an ObjcMethod object referencing one defined in the MachO.

+ (ObjcMethod *)methodFromStruct:(struct objc_method)method_struct inMachO:(MachO *)mach type:(enum method_type)type

Parameters

method_struct

the objc_method struct from the binary that defines one method.

mach

The MachO to search in.

type

The type of method that is being defined.

Return Value

An ObjcMethod object based on the information given.

Discussion

Parameters

mach

The MachO to search in.

offset

The virtual offset where the list of methods starts.

type

The type of method. Either, INSTANCE_METHOD, CLASS_METHOD, CAT_INSTANCE_METHOD or CAT_CLASS_METHOD.

Return Value

An array of all methods in the MachO.
