classInMachO:atVirtualOffset:

Create an ObjcClass object referencing one defined in the MachO.

+ (ObjcClass *)classInMachO:(MachO *)mach atVirtualOffset:(unsigned long)offset

Parameters

mach

The MachO to search in.

offset

The virtual offset the class is defined at.

Return Value

An ObjcClass located at the specified offset.

Discussion

This method is to be used with 32 bit binaries only.
